## ğŸš€ How It Works

### Getting Started
1. Start the app: `npm run dev`
2. Open: `http://localhost:3000`
3. Create your account with your name
4. Add a profile photo
5. Invite your friends!

### How to Add Friends
1. Go to "Explore"
2. Search for your friend's username
3. Click "Follow"
4. Start messaging!

### Creating Posts
1. Click "Create" button
2. Add photo and text
3. Write a caption
4. Post it!

### Sending Messages
1. Go to "Messages"
2. Search for your friend
3. Type your message
4. Send!

### Posting Stories
1. Click "Stories"
2. Add photo or video
3. It disappears in 24 hours
4. Friends can see who viewed it
